# ROS2 Hoverboard V2

## Overview
This project provides an interface for controlling a hoverboard using ROS2.

## Requirements
- ROS2 installed
- Required dependencies: [list any dependencies here]

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/mohamed4hanon/ROS2hoverboardV2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ROS2hoverboardV2
   ```
3. Build the project:
   ```bash
   ./build.sh
   ```

## Running the Project
To run the hoverboard control interface, execute:
```bash
./run.sh
```

## Code Structure
- **src/**: Contains source files.
- **scripts/**: Contains scripts for automation.

For detailed usage instructions, please refer to the respective script files: `build.sh` and `run.sh`.
